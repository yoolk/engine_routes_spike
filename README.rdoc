== Getting Started

1. Clone the repo, running bundle and migrating the database.
2. Add the seed data.
3. Start the application. Navigate to the home page http://localhost:3000. This should load the main apps home page.
4. Navigate to the products page: http://localhost:3000/products
5. Now terminate the application and update one of the InstantWebsite Domains to be 'localhost'
   InstantWebsite::Domain.last.update_attributes(name: "localhost")
6. Restart the application and now http://localhost:3000 will go to the InstantWebsite home page and http://localhost:3000/products will go to the InstantWebsite Products page
